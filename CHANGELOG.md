# Changelog

## 0.1.0 — 2025-03-19

Initial release.

- CLI chat REPL with streaming (Rich + prompt_toolkit)
- Folder, Gmail, Google Drive, Outlook ingestion
- ChromaDB vector store, hybrid retrieval, keyword fallback
- Entity extraction, relationship graph
- Agentic tools: draft email, summarize, find related, notes, reminders, calculator
- Briefing: stale leads, commitments, contracts
- Background sync daemon
- SSH auto-deploy
- HTTP API server (FastAPI)
- Security hardening (parameterized SQL, token permissions, input validation)
